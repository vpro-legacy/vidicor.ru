Необходимое оборудование

Для приема со спутника трансляции, производимой c помощью системы Vidicor, необходимо иметь компьютер с операционной системой MS Windows и следующее дополнительное оборудование:

·      компьютерный DVB-приемник в виде устанавливаемый в компьютер PCI-платы (напр., SkyStar-1, SkyStar-2), либо в виде внешнего блока с USB интерфейсом (напр., SkyStar USB), либо в виде внешнего блока с Ethernet-интерфейсом (напр., Pent@Office, ресивер DreamBox-500S),

·      стандартную спутниковую антенну («тарелку») достаточного диаметра (зависит от конкретного спутника и географического расположения точки приема) с установленным LNB (конвертором-«облучателем»).

LNB антенны соединяется специальным кабелем с DVB-приемником (компьютером).

После настройки системы принимаемую видеотрансляцию можно смотреть на экране монитора, подключенного к компьютеру. При необходимости использовать сигнал трансляции для передачи в телевизионный тракт следует обратиться в компанию Vidicor для консультации.

Настройка антенны

Антенна должна быть «нацелена» на спутник, находящийся в требуемой позиции. LNB должен быть повернут относительно своей оси в соответствии с позицией спутника и географическими  координатами точки приема (это важно! – неточная настройка может сделать прием невозможным). Эти настройки выполняются специалистом по установке спутниковых антенн либо самостоятельно – по процедурам, описанным в литературе.

Прием видеотрансляции

Прием видеотрансляции производится программой «Vidicor Watcher». Для этого необходимо выполнить следующее.

·      Загрузите программу установки «Vidicor Watcher» с сайта http://download.vidicor.ru.

·      Запустите загруженный файл и следуйте его инструкциям.

·      Запустите из папки «Vidicor Watcher», появившейся в меню «Пуск» («Start») à «Все программы» («All Programs»), программу «Satellite Watcher»).

При правильности всей предыдущей настройки, и наличии в данный момент видеотрансляции в окне программы «Vidicor Watcher» через несколько секунд появится видео, а в звуковой системе – звук.

При необходимости использовать сигнал трансляции для передачи в телевизионный тракт следует обратиться в компанию Vidicor для консультации.

Далее приводятся примеры настройки приема проводимой системой VIDICOR спутниковой трансляции для нескольких распространенных DVB приемников.

Пример настройки PCI DVB-приемника SkyStar1

Прежде всего, на компьютере должен быть установлен комплект программного обеспечения, прилагаемый к PCI DVB-приемнику SkyStar1 его изготовителем – драйвера и приложение «DVB Data».

Установка IP-адреса и маски подсети

·      Откройте «Пуск» («Start») à «Панель управления» («Control Panel») à «Сетевые подключения» («Network Connections») à «Соединение по локальной сети» («Local Area Connection»). Перейдите в режим обзора «Вид» («View») à «Таблица» («Details»)).

·      Откройте правой кнопкой мыши меню соединения устройства «Virtual DVB-S/-C/-T Network Adapter», откройте «Свойства» («Properties») à «Компоненты, используемые этим подключением» («This connection uses the following items») à «Протокол Интернет (TCP/IP)» («Internet Protocol (TCP/IP)») à «Свойства» («Properties») à «Использовать следующий IP адрес» («Use the following IP address»).

·      В поле «IP адрес» («IP address») введите нужный IP-адрес в виде 4-х десятичных чисел, разделенных точками (напр., «123.123.123.123»).

·      В поле «Маска подсети» («Subnet mask») введите маску подсети в виде 4-х десятичных чисел, разделенных точками. Установите ее значение «255.255.255.252».

·      Нажмите «OK» в текущем диалоге свойств протокола Интернет, Нажмите «Закрыть» («Close») в диалоге свойств сетевого подключения (предыдущий диалог).

Все открытые в ходе установки окна можно закрыть.

Установка параметров карты SkyStar1 программой «DVB Data»

·      Запустите из папки программ «Techno Trend» приложение «DVB Data».

·      В главном окне «Data Application» этого приложения выберите «Service selection» à «New».

·      В поле «Name of data service» введите имя «Vidicor», а в списке «Take settings from» выберите «Auto Tuning» и нажмите «OK».

·      В главном окне «Data Application» этого приложения выберите «Filters» à «Add». В соответствующих полях введите MAC-адрес (Relevant MAC bytes установить «6»), IP-адрес, PID. Установите флажок «Use this filter».

·      В главном окне «Data Application» приложения выберите «Setup» à «LNB/Satellite Settings». В появившемся диалоге «LNB/Satellite and DiSEqC Settings» укажите режим управления дополнительной спутниковой аппаратурой. Если используется одна постоянно наведенная на один спутник антенна с одним конвертером-облучателем режим должен быть установлен в «None». Проконсультируйтесь у специалиста, установившего ваше спутниковое оборудование.

·      Выберите в поле «LNB1» необходимый спутник (напр., «IntelSat 904»).

·      Нажмите кнопку «…» («Test satellite»). В поля «LOF 1», «Switch», «LOF 2» необходимо ввести значения соответствующих частот вашего LNB (конвертера-облучателя антенны), эти частоты указаны на корпусе LNB и в его паспорте (напр., LOF 1 = 9750, Switch = 11700, LOF 2 = 10600 МГц). Нажмите «OK» в диалогах «LNB Settings» и «LNB/Satellite and DiSEqC Settings».

·      В главном окне «Data Application» приложения выберите «Status/Tuner».

·      Выберите необходимый спутник из выпадающего списка «Select LNB/satellite».

·      Укажите частоту транспондера, символьную скорость, поляризацию в следующих полях.

·      Нажмите кнопку «Apply» («Применить»).

·      Убедитесь по индикаторам «Tuner state», что принимаемый сигнал достаточно сильный, имеет хорошее качество и что происходит синхронизация (появляется сигнал «Synchronization»).

Система настроена.

Наличие приема данных со спутника можно проконтролировать в закладке «Statistics» главного окна приложения «DVB Data». В списке появится введенный ранее PID, и будут меняться значения полей «Bytes» и «kbit/s» (объем принятых данных и скорость приема данных соответственно).

Пример настройки PCI DVB-приемника SkyStar2

Прежде всего, на компьютере должен быть установлен комплект программного обеспечения, прилагаемый к PCI DVB-приемнику SkyStar2 его изготовителем – драйвер и приложения «Setup4PC» и «Server4PC».

Установка MAC-адреса

·      Откройте «Пуск» («Start») à «Панель управления» («Control Panel») à «Система» («System») à «Оборудование» («Hardware») à «Диспетчер устройств» («Device Manager») à «Network adapters».

·      Откройте правой кнопкой мыши контекстное меню адаптера «TechniSat DVB-PC TV Star PCI», там откройте «Свойства» («Properties») à «Дополнительно» («Advanced») à «Свойство» («Property») à «Local network address» à «Значение» («Value»).

·      Введите в текстовое поле «Значение» («Value») нужный 12-значный MAC-адрес. Поле должно содержать только 12 шестнадцатеричных цифр (без «:», «-», пробелов, …; напр., «00D0D70D7BC4»).

·      Нажмите «OK» в текущем диалоге.

Открытые в ходе установки окна можно закрыть. Окно панели управления можно оставить для выполнения следующего пункта.

Установка IP-адреса и маски подсети

·      Откройте «Пуск» («Start») à «Панель управления» («Control Panel») à «Сетевые подключения» («Network Connections») à «Соединение по локальной сети» («Local Area Connection»).

·      Откройте правой кнопкой мыши контекстное меню соединения устройства «TechniSat DVB-PC TV Star PCI» (чтобы найти это соединение, возможно, придется переключить режим обзора «Вид» («View») à «Таблица» («Details»)), там откройте «Свойства» («Properties») à «Компоненты, используемые этим подключением» («This connection uses the following items») à «Протокол Интернет (TCP/IP)» («Internet Protocol (TCP/IP)»).

·      Откройте «Свойства» («Properties») à «Использовать следующий IP адрес» («Use the following IP address»).

·      В поле «IP адрес» («IP address») введите нужный IP адрес в виде 4-х десятичных чисел, разделенных точками (напр., «123.123.123.123»).

·      В поле «Маска подсети» («Subnet mask») введите маску подсети в виде 4-х десятичных чисел, разделенных точками. Установите ее значение «255.255.255.252»

·      Нажмите «OK» в текущем диалоге свойств протокола Интернет, Нажмите «Закрыть» («Close») в диалоге свойств сетевого подключения (предыдущий диалог).

Все открытые в ходе установки окна можно закрыть.

Установка параметров карты SkyStar2 программой Setup4PC из комплекта ее программного обеспечения

·      Запустите из папки программ «TechniSat DVB» приложение «Setup4PC».

·      В главном окне приложения укажите «Satellite Settings» à «Add».

·      В появившемся диалоге «Add Satellite» введите в поле «Satellite Name» название спутника (напр., «IntelSat 904»). В поля «LOF 1», «Switch», «LOF 2» необходимо ввести значения соответствующих частот вашего LNB (конвертера-облучателя антенны), эти частоты указаны на корпусе LNB и в его паспорте (напр., LOF 1 = 9750, Switch = 11700, LOF 2 = 10600 МГц). В выпадающем списке «DiSEqC» указывается режим управления дополнительной спутниковой аппаратурой. В большинстве случаев (когда у вас одна постоянно наведенная на один спутник антенна с одним конвертером-облучателем и без механического привода для «перенацеливания» на другие спутники) он должен быть установлен в «None». Точное значение этого режима может сказать специалист, установивший ваше спутниковое оборудование.

·      Нажмите в диалоге «Add Satellite» кнопку «OK».

·      В главном окне приложения укажите «Satellite Settings» à «Transponder Management» à «Add».

·      В появившемся диалоге «Add Transponder» выберите из выпадающего списка требуемый спутник (напр., «IntelSat 904»). В поле «Transponder» введите частоту приема (например, 11595 МГц), в поле «Symbol Rate» введите символьную скорость (напр., 29270 ксимв/с), в поле «FEC» выберите параметр избыточности кодирования (напр.,  7/8), в поле «Polarity» выберите поляризацию (напр., «Vertical/Right (Low)»).

·      Нажмите в диалоге «Add Transponder» кнопку «OK».

·      Нажмите в диалоге «Transponder Management» кнопку «Close».

·      В главном окне приложения введите «Satellite Settings» à «Data Services» à «Provider Name» à «Add».

·      В появившемся диалоге «Add Provider Name» введите в поле «Provider Name» имя «Vidicor» и нажмите кнопку «OK».

·      В диалоге «Data Services» в выпадающем списке «MAC Source» выберите значение «Broadband Receiver», при этом в поле «MAC Address» должен появиться тот самый MAC-адрес, который был установлен ранее.

·      В диалоге «Data Services» в блоке «PID List» снимите галочку «Hexadecimal» и в текстовом поле ниже ее введите нужный идентификатор приема PID, после чего нажмите кнопку «Insert».

·      В диалоге «Data Services» в блоке «Transponder» нажмите кнопку «Add».

·      В появившемся диалоге «Add Transponder» в выпадающем списке «Transponder» выберите частоту и поляризацию требуемого транспондера (напр., «11595 MHz: V/R»), в поле «Name» введите название спутника (напр., «IntelSat 904») и нажмите «OK».

·      Нажмите «OK» в диалоге «Data Services».

·      Прочитайте окно с сообщением о способе включения только что установленного спутникового сервиса и закройте его кнопкой «OK».

·      Закройте кнопкой «Close» окно приложения «Setup4PC».

·      В трее (область на панели задач рядом с часами) найдите иконку сервиса спутникового приемника – это цветной круг (в отсутствие принимаемого сигнала со спутника он красный, а при наличии приема – зеленый) с миниатюрным изображением спутника. Часть иконок в трее может быть скрыта; чтобы их увидеть, надо нажать кнопку со стрелкой с левого края трея (рядом с индикатором переключения клавиатурных раскладок).

·      Вызовите контекстное меню этой иконки (правой кнопкой мыши) и в нем выберите пункт «Vidicor :: IntelSat 904»).

Система настроена.

Проверить работоспособность можно следующим образом:

·      Наличие приема сигнала спутника можно проконтролировать в приложении «Setup4PC» в диалоге, вызываемом кнопкой «Status…» (там может потребоваться ввод информации о транспондере аналогично приведенному выше).

·      Наличие данных видеотрансляции в принимаемом спутниковом сигнале можно проконтролировать в этом же приложении в закладке «Statistics». В списке «Active PIDs» при наличии данных появится введенный ранее PID (в шестнадцатеричном виде). В списке фильтров данных (это главный список в закладке статистики) в строке, начинающейся на «UC» (сокращение от «Unicast») будут меняться значения полей «KBytes», «IP Packets», «Kbps» (объем принятых данных, количество принятых пакетов, скорость приема данных соответственно).

Пример настройки ресивера DreamBox-500S в качестве компьютерного DVB-приемника

Спутниковый ресивер DreamBox-500S можно применить в качестве компьютерного DVB приёмника, только если он имеет прошивку программных средств версии не ниже 1.8 от производителя. Версию прошивки можно узнать, указав команды «Dream» à «(8) Information» à «About» (после выбора необходимого нажимайте «OK»). При более ранней, чем 1.8,  версии прошивки или прошивки не от производителя ресивер необходимо перепрошить любым из методов, описанных в его руководстве.

Ресивер DreamBox-500S с одной стороны подключается к кабелю, идущему к LNB спутниковой антенны, с другой – к адаптеру LAN персонального компьютера, на котором будет идти просмотр. Подключение ресивера может производиться как к сетевому адаптеру компьютера («кроссовым» кабелем), так и через сетевой коммутатор в локальную сеть, с которой соединен компьютер. Заметим, что при подключении к сетевому коммутатору возможно осуществлять вещание в сети, когда прием возможен не только на любом компьютере сети, но и на любом количестве компьютеров одновременно.

Задание спутника и параметров транспондера

Параметры спутника, транспондера и канала приема (PID) могут быть установлены любым из способов, описанных в руководстве к ресиверу.

(Приведем для примера один из способов.

Укажите с пульта: «Dream» à (6)«Setup» à (2)«Service Searching» à (1)«Satellite Configuration»

Укажите конфигурацию антенной системы и выберите требуемый спутник в списке. Нажмите «Save».

В окне «Service Searching» выберите (4)«Transponder Edit». В появившемся диалоге «Transponders» выберите требуемый спутник. При необходимости – добавьте спутник. Выберите для этого спутника требуемый транспондер. При необходимости – добавьте транспондер, задав в соответствующем диалоге требуемые частоту транспондера, поляризацию, FEC, символьную скорость. Нажмите «Save».

В окне «Service Searching» выберите «Satfind» и убедитесь, что сигнал от спутника принимается с хорошим качеством – должно быть BER=0, должны выставиться флажки для «Lock» и «Sync». Если этого не происходит – антенна плохо спозиционирована, имеет недостаточный размер, кабель имеет слишком большое затухание либо неверно заданы параметры транспондера.

В меню окна «Service Searching» выберите «Manual Transponder Scan». В появившемся диалоге «Transponder Scan» выберите требуемый спутник, пропишите частоту транспондера, поляризацию, FEC, символьную скорость. Должны выставиться флажки «Lock» и «Sync». Нажмите кнопку «enter pids». Пропишите Name (имя сервиса), и Provider (имя провайдера), Service ID (например, задайте 0001). Остальные поля можно оставить без изменений. Нажмите «Save».

Настройка закончена.)

Настройка сетевого интерфейса ресивера

Настройка выполняется с помощью телевизора, подключенного к ресиверу, и пульта.

Укажите с пульта: «Dream» à (6)«Setup» à «Expert Setup» à «Communication Setup».

В окне «Communication Setup» отключите DHCP, пропишите IP-адрес ресивера (поле «IP»), маску подсети (поле «Netmask»), адрес DNS-сервера (поле «Nameserver») и шлюз (поле «Gateway»). Тип соединения – «LAN». Шлюзом будет являться тот компьютер, на котором будет идти просмотр видео. Настройки для примера: IP адрес 192.168.5.5, маска подсети 255.255.255.0, шлюз 192.168.5.1, DNS-сервер 0.0.0.0.

Данные сетевые настройки служат для связи между ресивером и принимающим (или управляющим через веб-страницу) компьютером и никак не связаны со спутниковым IP адресом получателя, указанным в параметрах трансляции!

Настройка сетевого интерфейса принимающего компьютера

Установите в сетевых настройках принимающего компьютера основной или дополнительный IP-адрес и маску подсети такие, чтобы компьютер и ресивер находились по этим параметрам в одной локальной сети. Настройки для примера (соответствуют приведенному выше примеру настроек ресивера): IP адрес 192.168.5.1, маска подсети 255.255.255.0, шлюз и DNS-сервер не задаются. Эти настройки также никак не связаны со спутниковым IP адресом получателя, указанным в параметрах трансляции.

Дальнейшие настройки ресивера можно при желании производить с компьютера через веб-страницу, используя имитатор пульта. Для этого откройте на принимающем компьютере интернет-браузер и введите в качестве адреса веб-страницы IP-адрес ресивера (в примере – 192.168.5.5); на запрос о пользователе и пароле введите «root» и «dreambox» соответственно (это исходные заводские установки, их можно изменить). Имитатор пульта запускается выбором «Control» à «Remote control».

Установка ресивера на нужный канал

Нажмите в главном окне ресивера на пульте кнопку переключения каналов (стрелки вверх, вниз) и в появившемся списке выберите нужные спутник и сервис (переключение между закладками производится цветными кнопками, соответствующими цветам закладок). На экране должна появиться информация о выбранном канале с названием сервиса.

Для контроля перезагрузите ресивер (из меню) и проверьте, на какой канал он установится после перезагрузки. Если всё сделано верно, то ресивер установится на этот же канал.

Установку на нужный канал можно произвести и программно или с помощью веб-интерфейса.

Настройка параметров DVB-потока

Создайте на компьютере файл «init» с содержимым

#!/bin/sh

IP=192.168.3.1

NETMASK=255.255.255.252

MAC=00:XX:XX:XX:XX:XX

PID=202

dvbnet $PID > /dev/null

ifconfig dvb0_0 $IP netmask $NETMASK hw ether $MAC arp mtu 1500

echo 1 > /proc/sys/net/ipv4/ip_forward

где приведенные для примера параметры заменены реальными указанный в параметрах трансляции:

·      IP – адрес DVB-интерфейса (это IP адрес получателя),

·      NETMASK – маска сети (ее значение 255.255.255.252),

·      MAC – требуемый МАС-адрес DVB-интерфейса,

·      PID – PID, выданный провайдером в шестнадцатеричном виде.

(Замечание. Концы строк в файле «init» должны задаваться ТОЛЬКО кодами <0Ah>, а не парами <0Dh><0Ah>, как в MS Windows!)

Скопируйте этот файл «init» ресивер в папку ftp://<user>:<password>@<IP-адрес ресивера>/var/etc/ (исходно user: «root», password: «dreambox»), используя любой ftp-менеджер (например, с помощью распространенного менеджера FAR).

Установите с помощью файл-менеджера для скопированного файла «init» права доступа «rwx r-x r-x» (код 755).

Система настроена, ее следует перезагрузить (указать на ресивере Control à Reboot).

Для окончательной проверки можете проверить, как отработал init-скрипт, соединившись с ресивером телнетом (набрав «telnet <IP-адрес ресивера>» и набрав ifconfig; в ответном сообщении должны быть перечислены три интерфейса: dvb0_0, eth0 и lo.
